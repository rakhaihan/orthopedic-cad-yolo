# Data Dictionary

- image_id: unique id or filename without extension
- file_name: image filename (e.g., IMG0001.jpg)
- fractured: 0/1 label for fracture presence
- fracture_c: fracture class/type (integer or string)
- hand, leg, hip, shoulder, mixed: binary flags for anatomical region
- hardware: presence of implant (0/1)
- multiscan: multiple exposures (0/1)
- frontal, lateral, oblique: view orientation flags
